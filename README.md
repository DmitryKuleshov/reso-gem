# Reso
Ruby implementation of the RESO Web API

## Usage
Install and use.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'reso'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install reso
```
